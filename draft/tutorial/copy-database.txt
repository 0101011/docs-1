
===============
 Copy Database
===============

Sypnosis
--------

This document oulines the procedure to copy MongoDB databases from one
server to another. 

There are special commands to copy a MongoDB database from one server to
another. This is faster than using :program:`mongodump` and
:program:`mongorestore` programs as there is no intermediate files to
be created.

--------

- does not lock
