============================
Installing MongoDB
============================

MongoDB runs on most platforms on both 32-bit and 64-bit architectures.
`10gen <http://www.10gen.com>`_, the makers of MongoDB, provides both binaries
and packages. Choose your platform below:

* :doc:`RedHat Enterprise, CentOS, and Fedora Linux <tutorial/install-mongodb-on-redhat-centos-or-fedora-linux>`
* :doc:`Debian and Ubuntu <tutorial/install-mongodb-on-debian-or-ubuntu-linux>`
* :doc:`Linux and Unix <tutorial/install-mongodb-on-linux>`
* :doc:`OS X<tutorial/install-mongodb-on-os-x>`
* :wiki:`Windows <Quickstart+Windows>`

Release Notes
~~~~~~~~~~~~~

You should always install the latest, stable version of MongoDB.
Stable versions are indicated by an even-numbered minor version
number. For example, v2.0 and v2.2 are stable, while v2.1 is a
development version.

- Current Stable Releases

  - :wiki:`v2.0 Release Notes <2.0+Release+Notes>`
