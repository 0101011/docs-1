===
$ne
===

.. default-domain:: mongodb

.. operator:: $ne

   *Syntax*: ``{field: {$ne: value} }``

   :operator:`$ne` selects the documents where a ``field`` is not equal
   (i.e. ``!=``) to the specified ``value``. This includes documents
   that do not contain the ``field``.
      
   For example, you would query:
   
   .. code-block:: javascript

      db.inventory.find( { qty: { $ne: 20 } } )

   to select all documents in ``inventory`` where ``qty`` does not equal
   ``20``, including those documents that do not contain ``qty``. 
   
   You may use the :operator:`$ne` operator to select fields from
   embedded documents. For example, you would query:
 	  
   .. code-block:: javascript

      db.inventory.update( { "carrier.state": { $ne: "NY" } }, { $set: { qty: 20 } } )
   		
   to update a single document in inventory where the the field ``state``
   from the embedded document ``carrier`` does not equal "NY" or where the
   the field does not exist.
   
   .. seealso::
   
      method:`find() <db.collection.find()>`, :method:`update()
      <db.collection.update()>`, :operator:`$set`.
