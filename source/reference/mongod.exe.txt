============================
:program:`mongod.exe` Manual
============================

Sypnosis
--------

:program:`mongod.exe` is the application for the MongoDB
system on the Windows platform. It is equivalent to :program:`mongod`
on other platforms with additional options to interoperate on the
Windows platform.

This document will explain all additional options for
:program:`mongod.exe` not found in :program:`mongod`. Please refer to
the :doc:`/reference/mongod` for main functionality.

To install and use :program:`mongod.exe` please read this
:doc:`document</tutorial/install-mongodb-on-windows>`.


Options
-------

.. program:: mongod.exe

.. option:: --install

   This installs mongod.exe as a Windows Service and exits.

.. option:: --remove

   This removes mongod.exe as a Windows Service. If mongod.exe is
   running, it will be stopped then removed as a service.

   Requires: --serviceName (if set differently than default)


.. option:: --reinstall

   This removes mongod.exe and reinstalls mongod.exe as a Windows
   Service. 


.. option:: --serviceName <name>

   *Default*: MongoDB

   *Requires*: :option:`--install` or :option:`--remove`

   This will set the service name of mongod.exe when running as a
   Windows Service. This is the name to be used with the ``net start
   <name>`` and ``net stop <name>`` commands.


.. option:: --serviceDisplayName <name>

   *Default*: Mongo DB

   *Requires*: :option:`--install`

   This will set the name listed on the Services administrative
   application


.. option:: --serviceDescription <description>

   *Default*: MongoDB Server 

   *Requires*: :option:`--install`

   This sets the mongod.exe service description. 

   .. note:: 
      
      For long descriptions with spaces, you must enclose the
      description within quotes


.. option:: --serviceUser <user>

   *Requires*: :option:`--install`

   Runs the mongod.exe service in the context of a certain user. This
   user must have "Log on as a service" privileges. 


.. option:: --servicePassword <password>

   *Requires*: :option:`--install`

   This sets the password for <user> for mongod.exe when running with
   --serviceUser option.
