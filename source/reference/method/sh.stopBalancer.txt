Definition
----------

.. method:: sh.stopBalancer(timeout, interval)

   Disables the balancer in a sharded cluster and waits for balancing to
   complete.

   .. important::

      You can only run :method:`sh.stopBalancer()` on a
      :program:`mongos` instance. :method:`sh.stopBalancer()`
      errors if run on :program:`mongod` instance.
