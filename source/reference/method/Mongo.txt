=======
Mongo()
=======

.. default-domain:: mongodb

.. method:: Mongo()

   JavaScript constructor to instantiate a database connection from the
   :program:`mongo` shell or from a JavaScript file.

   The constructor can take:

   - no parameters to instantiate a connection to the localhost
     interface on the default port:

     .. code-block:: javascript

        new Mongo()

   - a string ``<host>`` parameter to instantiate a connection to the
     ``<host>`` and the default port:

     .. code-block:: javascript

        new Mongo(<host>)

   - a string ``<host:port>`` parameter to instantiate a connection to
     the ``<host>`` and the ``<port>``:

     .. code-block:: javascript

        new Mongo(<host:port>)

   .. seealso:: :method:`Mongo.getDB()`
