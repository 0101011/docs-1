=========================
copydbgetnonce (internal)
=========================

.. default-domain:: mongodb

.. dbcommand:: copydbgetnonce

   Client libraries use :dbcommand:`copydbgetnonce` to get a one-time
   password for use with the :dbcommand:`copydb` command.
