===============
Read Operations
===============

.. default-domain:: mongodb

Read operations, or *queries*, retrieve data stored in the database.
This section describes and provides examples of read operations in
MongoDB. The section also describes the factors that affect the read
operations. All of the examples in this section use the
:program:`mongo` shell interface. To see the equivalent in a particular
:doc:`MongoDB Driver </applications/drivers>`, see your :api:`driver
API documentation <>`. MongoDB also provides a rich library of
:doc:`aggregation </aggregation>` operations to perform various
aggregation tasks. For these operations, refer to the :doc:`aggregation
</aggregation>` section.
