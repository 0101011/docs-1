===========
Insert Data
===========

.. default-domain:: mongodb

Overview
--------

Use the :method:`~db.collection.insert()` method to add documents to a
:term:`collection` in MongoDB. If you attempt to add documents to a
collection that does not exist, MongoDB will create the collection for
you.

-------------

.. include:: /includes/examples/example-gs-insert-prereq.rst

Procedures
----------

Insert a Document
~~~~~~~~~~~~~~~~~

.. include:: /includes/examples/example-gs-insert.rst

The returned document shows that MongoDB added an ``_id`` field to the
document. The :manual:`ObjectId </reference/object-id>` values in your
documents will differ from the ones shown.
 
Insert Multiple Documents
~~~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/examples/example-gs-insert-array.rst

The :data:`~BulkWriteResult.nInserted` field specifies the number of
documents inserted. If the operation encounters an error, the
:method:`BulkWriteResult` object will contain information regarding
the error.

You can also verify the insert operation by :doc:`querying </query>`
the collection. The inserted documents will each have an ``_id``
field added by MongoDB.
  
Additional Information
----------------------

You can also use the :method:`Bulk()` API to insert multiple documents.
See :method:`Bulk()` for more information.
