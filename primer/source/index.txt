============================
Getting Started with MongoDB
============================

.. default-domain:: mongodb

Overview
--------

Welcome to the MongoDB Primer, a guide for getting started with
MongoDB. The Primer outlines the process for installing MongoDB and
gets you started using MongoDB.

The Primer provides a tightly focused introduction to MongoDB. All of
the examples in this version of the Primer use the :doc:`/client`;
however, the Primer is available in :ref:`other editions
<primer-other-editions>` for various drivers.
